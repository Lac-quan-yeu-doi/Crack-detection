# Concrete Crack Detection using Classical Image Processing

A simple crack detection project on concrete surfaces using only traditional computer vision techniques.

This repository compares several classic methods (Sobel, Morphological operations, Bottom-hat, Canny, Background suppression, etc.) to detect cracks on concrete surface.

## Features
- Pure OpenCV + NumPy + scikit-image
- No machine learning / neural networks
- Step-by-step visualizations
- All results automatically saved in `/output/`
- Fully reproducible

## Requirements
- Python 3.11+
- OpenCV, NumPy, Matplotlib, scikit-image, SciPy
